## NOTE — index.css hex custom properties
`src/index.css` defines `--state-answered-bg`, `--state-flagged-*`, `--highlight-bg` etc. as hex values in `:root` and `[data-high-contrast]` blocks. These are intentional state-color custom properties for the exam environment and are **not** violations — they're defining tokens, not bypassing them. The dark/HC variants (`#003300`, `#00FF00`, `#FFFF00`) exist for accessibility mode. No action needed on the CSS custom property definitions themselves.
