## HIGH — Banned CSS patterns

### `color-mix(in oklch, ...)` (173 instances)
The DS anti-pattern blacklist bans `color-mix()` for `background`/`backgroundColor` inline styles. This is by far the largest violation category — nearly every entity page uses it for active-state styling. Replace with DS token pairs that have known contrast:
- Active tint: `var(--brand-tint)` background + `var(--brand-color)` text
- Muted states: `var(--muted)` background + `var(--muted-foreground)` text

### `uppercase tracking-wide` / `tracking-wider` (79 instances)
Banned per Mobbin-first anti-pattern blacklist. Replace with DS typography: `text-xs font-medium text-muted-foreground` for section labels.
