# PCE — Research Insights

Distilled insights from rr-insights, customer interviews, ethnography, surveys.

Same shape as `apps/exam-management/docs/research/insights/`. See that README for the full convention.

## Quick reference

| Artifact | Path |
|---|---|
| Single insight | `<YYYY-MM-DD>-<slug>.md` |
| Cross-participant theme | `themes/<slug>.md` |
| Skill that handles routing | `.claude/skills/research-intake/SKILL.md` |

## PII rule

**Participant IDs only.** No names, no emails, no identifying details.
